Objetivos

Creación de un sitio web en el cual se podrán visualizar de manera rápida y sencilla todos los platos disponibles, junto a su descripción e información.

El sitio será adaptable a multiples dispositivos, tendrá animaciones, formularios de contacto, una sección de información de la empresa, de contacto, y finalmente la información especifica de los platos.

Detalles del Proyecto

¿Que haremos?

  1. Diseño

    Se realizara un diseño preliminar del sitio web, el cual se realizara teniendo en cuenta los deseos del cliente, las posibilidades técnicas, y nuestras recomendaciones y experiencias.

  2. Maquetación

    Aquí comenzara el proceso de desarrollo del sitio web, primero crearemos las estructuras básicas que tendrá el sitio web, para luego poder continuar con una base solida.

  3. Estilos

    Una vez tengamos las estructuras básicas, podremos comenzar a darle un mejor look and feel a nuestro sitio web, logrando asi que sea mucho mas moderno y atractivo para los clientes.

  4. Pruebas y conformidad

    Cuando las estructuras y estilos estén finalizados, realizaremos pruebas, para controlar el correcto funcionamiento y aspecto del sitio.

  5. Subida al Servidor y entrega

    En esta etapa del proyecto subiremos el sitio a un hosting, y este sera configurado para que finalmente pueda ser accesible por los clientes.

¿Que recursos usaremos?

Para el desarrollo

  • HTML para la estructuras
  • Bootstrap para el maquetado.
  • CSS y SASS para los estilos del sitio.
  • Google Fonts para las fuentes.
  • Font Awesome para la iconografia.

Para la puesta en producción

  • Utilizaremos un dominio para tener un facil acceso al sitio
  • Un hosting dedicado en donde poder almacenar y desde el cual poder administrar el sitio.
